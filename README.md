# Markdown编辑器 (构建版本)

这是Markdown编辑器的优化构建版本，所有代码已经过压缩和优化。

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 或者通过HTTP服务器提供服务（推荐）:
   - 使用Node.js: `npx serve` 或 `npx http-server`
   - 使用Python: `python -m http.server`
